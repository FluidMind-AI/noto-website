## Local Development

```bash
# Clone the repo
git clone git@github.com:FluidMind-AI/noto-website.git
cd noto-website

# Open in browser
open index.html
```

## Deployment

This site is automatically deployed via GitHub Pages when changes are pushed to `main`.

## Tech Stack

- Static HTML/CSS/JS
- Tailwind CSS (via CDN for some pages)
- Google Fonts (DM Sans, Poppins)
